# この記事の動機

テストケースの増加に伴って、CIの実行時間が長くなっていました。CIの実行時間が長いと、フィードバックを得られるまでの時間が長くなり、開発効率が低下します。

弊社ランプでは、開発効率の向上のため、継続的にCIの改善に取り組んでいます。

本記事では、テストの実行時間を短縮するために、PHPUnitでプロファイラを実装し、ボトルネックとなっているテストケースを特定できた方法を紹介します。

結果として、CIのテスト実行時間を約8分から約5分に短縮することができました。

# 推測するな、計測せよ

パフォーマンスチューニングの世界では、よく「推測するな、計測せよ（Don't guess, measure）」という格言が引用されます。システムのボトルネックを特定するためには、実際のデータに基づいて判断することが重要です。

CIの改善も例外ではありません。ボトルネックになっているところ以外を闇雲に最適化しても、効果は限定的です。生成AIに修正を依頼するにも膨大なテストケースの中から、どのテストケースを改善すべきかを探させるのはコンテキストが収まらず困難です。

まずは、どのテストが時間を要しているのかを計測することから始めましょう。

# プロファイラの実装

実装には、PHPUnitのテストフック機能を利用します。テストフックを使うことで、各テストの開始時と終了時に処理を挟むことができます。

```php:tests/TestTimeLogger.php
<?php

namespace Tests;

use PHPUnit\Runner\AfterLastTestHook;
use PHPUnit\Runner\AfterTestHook;
use PHPUnit\Runner\BeforeTestHook;

/**
 * A PHPUnit test hook that logs the execution time of each test.
 */
class TestTimeLogger implements AfterLastTestHook, AfterTestHook, BeforeTestHook
{
    /**
     * @var array
     */
    protected array $times = [];

    /**
     * @var float
     */
    protected float $test_start_time;

    /**
     * A test started.
     */
    public function executeBeforeTest(string $test): void
    {
        $this->test_start_time = microtime(true);
    }

    /**
     * A test ended.
     */
    public function executeAfterTest(string $test, float $time): void
    {
        [$class, $test_name] = explode('::', $test, 2);
        $this->times[$class][$test_name] = microtime(true) - $this->test_start_time;
    }

    /**
     * All tests ended.
     */
    public function executeAfterLastTest(): void
    {
        if (! empty($this->times)) {
            $this->printResults();
        }
    }
    /**
     * Print the test execution times.
     */
    protected function printResults(): void
    {
        // 好きなようにフォーマットして出力する
    }
}

```

## 出力例

```php
    protected function printResults(): void
    {
        // Print the top 20 slowest tests overall
        echo "Top 20 Slowest Tests:\n";
        echo "-------------------\n";
        $all_tests = [];
        foreach ($this->times as $class => $tests) {
            $short_class = preg_replace('/^.*\\\\/', '', $class);
            foreach ($tests as $test => $time) {
                $all_tests[$short_class.'::'.$test] = $time;
            }
        }
        arsort($all_tests);
        $count = 0;
        foreach ($all_tests as $test => $time) {
            echo $test.': '.number_format($time, 2)." seconds\n";
            $count++;
            if ($count >= 20) {
                break;
            }
        }
    }
```
